# FractureOrSprain? — AI-Powered Clinical Triage App

A medical triage web application designed for instant injury assessment and X-Ray analysis.

## Features
- **Symptom Assessment**: Clinical triage based on pain intensity, mobility loss, swelling, and deformity.
- **X-Ray Analysis**: Deep learning visual assessment for detection of fractures vs. sprains.
- **AI Doctor**: Conversational assistant powered by Gemini 2.5 Flash for medical guidance.
- **Multilingual Support**: Real-time translation supporting English, Hindi, Telugu, Tamil, and Marathi.
- **OLED Dark / Light Theme**: Dynamic true-black dark mode toggle with smooth animations.
- **History Tracking**: Secure patient dashboard integrated with Firebase Firestore.
